Now earn money effortlessly through your own affiliate link. INRDeals is an online and offline platform that offers deals and discounts at most trusted and popular online and retail stores. == Installation ==

This is how you can install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins directory directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make an account on INRDeals (if not already signed up).
4. Add Your INRDeals Username. Your Username can be added to Dashboard< Settings< INRDEALS Settings.
